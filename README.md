# zonedDateTime
В методе main выведи на экран переменную zonedDateTime в таком виде:
4 9.3.19 06:03:07.319180500 Europe/Kiev

Где:

4 — номер дня недели, т.е. четверг;
9 — день месяца;
3 — месяц;
19 — год;
06 — часы;
03 — минуты;
07 — секунды;
319180500 — наносекунды;
Europe/Kiev — временная зона.


псевдокод:

```
ПСЕВДОКОД:

ПРОГРАММА ФорматированиеДатыВремени

ИМПОРТИРОВАТЬ ZonedDateTime
ИМПОРТИРОВАТЬ DateTimeFormatter

ПЕРЕМЕННАЯ:
    zonedDateTime = ZonedDateTime.текущееВремя()  // Получаем текущие дату и время с часовым поясом

ФУНКЦИЯ main(аргументы):
    // Создаем форматтер с шаблоном для отображения:
    // e - день недели (1-7)
    // d - день месяца
    // M - месяц
    // y - год
    // HH - часы (00-23)
    // mm - минуты
    // ss - секунды
    // n - наносекунды
    // VV - идентификатор часового пояса
    dateTimeFormatter = DateTimeFormatter.создатьФормат("e d.M.yy HH:mm:ss.n VV")
    
    // Форматируем дату-время в строку по заданному шаблону
    text = dateTimeFormatter.форматировать(zonedDateTime)
    
    // Выводим результат
    ВЫВЕСТИ text

КОНЕЦ_ПРОГРАММЫ
```

Пояснение шаблона формата:

· e - день недели (1 = понедельник, 7 = воскресенье)
· d.M.yy - день.месяц.год (двухзначный)
· HH:mm:ss - часы:минуты:секунды (24-часовой формат)
· n - наносекунды
· VV - идентификатор часового пояса (например, "Europe/Moscow")

Пример вывода:
3 15.11.23 14:30:45.123456789 Europe/Moscow
